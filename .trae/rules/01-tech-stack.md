# 技术栈

## 后端
- Spring Boot 3.2 + Java 17
- ORM：MyBatis-Plus 3.5
- 数据库：MySQL 8.0
- 缓存：Redis
- 认证：JWT + Spring Security
- 日志：@Slf4j

## 前端
- Vue 3 + Composition API + TypeScript
- UI：Element Plus
- 状态管理：Pinia
- HTTP：axios，封装在 api/request.ts 中
- 路由：Vue Router
- 全局样式：Element Plus 提供的全局样式
- 其他：Element Plus 提供的其他组件
- 代码规范：ESLint + Prettier
